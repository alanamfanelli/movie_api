myFlix App - Documentation

This is my movie app where the user can find information about movies.

The table below describes each endpoint of this API and how to access it.

Description URL HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object holding data about all movies
Get data about a single movie by title /movies/[Title] GET None JSON object holding data about a single movie, containing title, description, genre, and director. Example:

{ "Genre": { "Name": "Thriller", "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involvesexcitement and suspense in the audience." }, "Director": { "Name": "Jonathan Demme", "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.", "Birth": "1944", "Death": "2017" }, "Actors": [], "_id": "5d9bab4307ab30237c44c353", "Title": "Silence of the Lambs", "Description": "A young FBI cadet must receive the help of an incarcerated and manipulative cannibal killer to help catch another serial killer.", "ImagePath": "silenceofthelambs.png", "Featured": true }

Get data about a movie genre /movies/genres/[Name] GET None JSON object holding data about a movie genre, containing name and description. Example:

{ "Name": "Action", "Description": "Action film is a film genre in which the protagonist or protagonists are thrust into a series of events that typically include violence, extended fighting, physical feats, and frantic chases." }

Get data about a director /movies/directors/[Name] GET None JSON object holding data about a director, containing name, bio, birth year, death year. Example:

{ "Name": "Jonathan Demme", "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.", "Birth": "1944" }

Add a new user to the registry. /users POST JSON object holding data about the user to add, structured like:

{ "Username" : "helloworld123456", "Password" : "password123456", "Email" : "example123456@gmail.com", "Birthday" : "1970-03-21" }

JSON object holding data about the user that was add, structured like:

{ "FavoriteMovies": [], "_id": "5d9d82069bc166069f68524d", "Username": "helloworld123456", "Password": "password123456", "Email": "example123456@gmail.com", "Birthday": "1970-03-21T00:00:00.000Z", "__v": 0 }

Update a user's information /users/[Username] PUT A JSON object holding data about the user to add, structured like:

{ "Username" : "Newusername1231561", "Password" : "Newpassword3486198", "Email" : "Newemail16513651@gmail.com", "Birthday" : "1980-10-15" }

A text message indicating the user's information was updated.
Add a movie to the user's favorites list /users/[Username]/Movies/[MovieID] POST None A text message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /users/[Username]/Movies/[MovieID] DELETE None A text message indicating the movie title was deleted from the user's favorites list.
Remov a user from the registry. /users/[Username] DELETE None A text message indicating the user was removed from the registry.
